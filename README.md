# torshammer

## This is a maintained fork of Torshammer

## Usage on Linux
```sh
git clone https://github.com/lidarbtc/torshammer.git

pip install pysocks

```

## Example
```sh
Use command line : python torshammer.py <target> <thread> <viator>
      └──────────> python torshammer.py https://example.com 250 -T
```
